查看项目历史版本

git reflog

根据序列号前进或回退版本

git reset --hard [序列号]

查看文件内容

cat [文件名]

查看分支情况

git branch -v

查看项目当前状态

git status

创建分支

git branch [分支名]

切换分支

git checkout [分支名]

添加到本地库

git add [文件名]

提交到远程库

git commit -m "修改描述" [文件名]

合并分支，先切换至待合并分支上，再merge合并分支对象

git checkout [待合并分支名]
git merge [分支名]

创建文件夹

mkdir [文件名]

初始化仓库，进入文件夹后

git init

查看git用户与邮箱

git config user.name 
git config user.email

配置公钥以后，与gitee添加联系

git remote add gitee git@gitee.com:xusuona/spaims.git

git关联之后提交整体步骤

git add .
git commit -m 提交介绍
git push gitee master

期间还可不间断地查看状态

git status

若远程库中留有无关文件，但本地已提交，则在本地删除该已提交文件

git  rm -r --cached [文件名]
